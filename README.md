# TinDog 🐾

A responsive landing page for a fictional dog-matching app, built with Bootstrap 5. Completed as part of the Bootstrap section of Angela Yu's Full Stack Web Development Bootcamp.

> All Bootstrap components were identified and integrated independently — no hints or starter templates used.

---

## 🚀 Live Demo

[View on GitHub Pages](https://sakshamagrawalcode-cpu.github.io/copy-website-by-bootstrap/)

---

## 🛠️ Built With

- HTML5
- CSS3
- [Bootstrap 5.3.8](https://getbootstrap.com/)

---

## 📄 Sections

| Section | Description |
|---|---|
| **Hero** | Full-width intro with gradient background, app download buttons, and phone mockup |
| **Features** | Three-column icon grid highlighting app benefits |
| **Testimonial** | Pull quote with press logo strip (TechCrunch, Mashable, etc.) |
| **Pricing** | Three-tier pricing cards (Chihuahua / Labrador / Mastiff) |
| **Footer** | Multi-column nav links with gradient background |

## 🧠 What I Learned

- Bootstrap grid system — `container`, `row`, `col-*` and responsive breakpoints
- Margin and padding utility classes (`m-*`, `p-*`, `mx-*`, `py-*`, etc.)
- Typography and flexbox alignment utilities
- Pre-built Bootstrap components: cards, buttons, pricing tables
- Layering custom CSS on top of Bootstrap without conflicts
- Building a multi-section responsive layout from scratch
